================================================================================
                   COMMON ISSUES & SOLUTIONS
================================================================================

ISSUE: "Cannot POST /api/teacher-requests/submit"
  → Check: Is backend running? Port 5918?
  → Check: Is route registered in server.js?
  → Check: Is middleware in correct order?

ISSUE: Form doesn't show after registration
  → Check: Is Router configured with /teacher-request path?
  → Check: Are imports correct in Router.jsx?
  → Check: Browser console for JS errors

ISSUE: Approve button doesn't work
  → Check: Are you logged in as admin?
  → Check: Is API call reaching backend? (Check logs)
  → Check: Is database connection working?

ISSUE: Auto-refresh not working on status page
  → Check: Is interval being set?
  → Check: Is useCallback being used correctly?
  → Check: Are dependencies correct?

ISSUE: Still can access features after rejection
  → Check: Is auth middleware checking approvalStatus?
  → Check: Is User.approvalStatus set to "REJECTED"?
  → Check: Are protected routes using middleware?
